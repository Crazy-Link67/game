# Journey to the West Offline PWA Game Plan

Implemented as a buildless vanilla web app with `index.html`, `styles.css`, `data.js`, `app.js`, `sw.js`, `manifest.webmanifest`, and `assets/compass.svg`.

V1 includes the full data-driven engine, six heroes, six uniquely named 500-section books, local saves, money tracking, settings, maps, one max-out codeword, temporary d6 stat boosts for blocked actions, undo for game actions, online-mode leaderboard/chat surfaces, stat upgrades, and offline caching.
