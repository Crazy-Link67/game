# Journey to the West: Frontier Trails Offline PWA Game Plan

Implemented as a buildless vanilla web app with `index.html`, `styles.css`, `data.js`, `app.js`, `sw.js`, `manifest.webmanifest`, and `assets/compass.svg`.

V1 includes the full data-driven engine, six Wild West heroes, custom/random character names, weapons and mounts, eleven uniquely named 1,000-section frontier books, local saves, export/import saves, money tracking, settings, book-specific world maps, boxed numbered section routes, one max-out codeword, temporary d6 stat boosts for blocked actions, undo for game actions, online-mode leaderboard/chat surfaces, stat upgrades, a compass-and-crossed-pistols logo, paper-style story screens, and offline caching.
